تحلیل احساسات با CommentAnalyzer

این کد از کلاس CommentAnalyzer برای تحلیل احساسات متن استفاده می‌کند و نتیجه را به صورت recommended یا not_recommended برمی‌گرداند.
# کد پایتون واقعی - تحلیل با CommentAnalyzer from PerSent import CommentAnalyzer analyzer = CommentAnalyzer() analyzer.loadModel() text = "متن ورودی کاربر" result = analyzer.analyzeText(text) print(f"Sentiment: {result}")

تحلیل احساسات با SentimentAnalyzer

این کد از کلاس SentimentAnalyzer برای تحلیل احساسات متن استفاده می‌کند و درصد هر یک از احساسات را نمایش می‌دهد.
# کد پایتون واقعی - تحلیل با SentimentAnalyzer from PerSent import SentimentAnalyzer analyzer = SentimentAnalyzer() analyzer.loadModel() sample_text = "متن ورودی کاربر" result = analyzer.analyzeText(sample_text) for emotion, score in sorted(result.items(), key=lambda x: x[1], reverse=True): print(f"{emotion}: {score:.2f}%")

نصب و تست کتابخانه PerSent

این کد کتابخانه PerSent را نصب کرده و یک تست کامل انجام می‌دهد.
# کد کامل برای نصب و تست PerSent import micropip # نصب کتابخانه PerSent await micropip.install('persent') from PerSent import CommentAnalyzer, SentimentAnalyzer # تحلیل با CommentAnalyzer comment_analyzer = CommentAnalyzer() comment_analyzer.loadModel() # تحلیل با SentimentAnalyzer sentiment_analyzer = SentimentAnalyzer() sentiment_analyzer.loadModel() text = "متن ورودی کاربر" print("=== تحلیل با CommentAnalyzer ===") comment_result = comment_analyzer.analyzeText(text) print(f"نتیجه: {comment_result}") print("\n=== تحلیل با SentimentAnalyzer ===") sentiment_result = sentiment_analyzer.analyzeText(text) for emotion, score in sorted(sentiment_result.items(), key=lambda x: x[1], reverse=True): print(f"{emotion}: {score:.2f}%")
