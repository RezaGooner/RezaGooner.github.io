اجرای کدهای پایتون برای تست کتابخانه PerSent

توجه: این صفحه برای شبیه‌سازی اجرای کدهای پایتون مربوط به کتابخانه PerSent طراحی شده است. در محیط واقعی، این کدها نیاز به نصب کتابخانه و اجرا در محیط پایتون دارند.

تحلیل احساسات با CommentAnalyzer

این کد از کلاس CommentAnalyzer برای تحلیل احساسات متن استفاده می‌کند و نتیجه را به صورت recommended یا not_recommended برمی‌گرداند.
pip install persent from PerSent import CommentAnalyzer analyzer = CommentAnalyzer() analyzer.loadModel() text = input() result = analyzer.analyzeText(text) print(f"Sentiment: {result}") # Output: Sentiment: recommended

تحلیل احساسات با SentimentAnalyzer

این کد از کلاس SentimentAnalyzer برای تحلیل احساسات متن استفاده می‌کند و درصد هر یک از احساسات را نمایش می‌دهد.
from PerSent import SentimentAnalyzer analyzer = SentimentAnalyzer() analyzer.loadModel() sample_text = input() result = analyzer.analyzeText(sample_text) for emotion, score in sorted(result.items(), key=lambda x: x[1], reverse=True): print(f"{emotion}: {score:.2f}%")

مثال‌های بیشتر برای تست

در این بخش می‌توانید متن‌های مختلفی را برای تست کتابخانه وارد کنید.
# کد کامل برای تحلیل احساسات pip install persent from PerSent import CommentAnalyzer, SentimentAnalyzer # تحلیل با CommentAnalyzer comment_analyzer = CommentAnalyzer() comment_analyzer.loadModel() # تحلیل با SentimentAnalyzer sentiment_analyzer = SentimentAnalyzer() sentiment_analyzer.loadModel() text = input("لطفاً متن فارسی خود را وارد کنید: ") print("=== تحلیل با CommentAnalyzer ===") comment_result = comment_analyzer.analyzeText(text) print(f"نتیجه: {comment_result}") print("\n=== تحلیل با SentimentAnalyzer ===") sentiment_result = sentiment_analyzer.analyzeText(text) for emotion, score in sorted(sentiment_result.items(), key=lambda x: x[1], reverse=True): print(f"{emotion}: {score:.2f}%")
